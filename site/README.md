
# MyGrade Calculator (나이스 내신 산출기)

이 프로젝트는 학생의 나이스(NEIS) 생활기록부 HTML 파일을 업로드하여, 내신 등급과 연세대학교 교과전형 환산점수를 자동으로 계산해주는 웹 애플리케이션입니다.

## 🚀 기능 특징

1.  **프라이버시 보호**: 모든 분석은 브라우저 내(Client-Side)에서 이루어집니다. 개인 생기부 파일이 서버로 전송되지 않습니다.
2.  **자동 파싱**: 복잡한 나이스 HTML 구조를 자동으로 해석하여 성적 데이터를 추출합니다.
3.  **연세대식 산출**: Z점수, 석차등급, 이수단위, 진로선택 가산점, 감점 등을 모두 고려한 정교한 계산 로직이 포함되어 있습니다.

## 📂 폴더 구조

```
site/
├── css/
│   └── style.css       # 프리미엄 다크 테마 디자인
├── js/
│   ├── main.js         # UI 인터랙션 및 파일 처리
│   ├── parser.js       # 나이스 HTML 파싱 로직
│   └── calculator.js   # 내신 산출 알고리즘 (Python 로직 이식됨)
└── index.html          # 메인 실행 파일
```

## 🌐 호스팅 및 배포 가이드

이 사이트는 **정적 웹사이트(Static Website)**입니다. 별도의 백엔드 서버(Python, Node.js 등)가 필요 없으며, HTML/CSS/JS 파일만 호스팅하면 됩니다.

### 추천 호스팅 서비스
1.  **Vercel** (추천): 가장 쉽고 빠릅니다. GitHub에 코드를 올리고 Vercel과 연동하면 끝입니다.
2.  **Netlify**: Vercel과 유사하며 drag-and-drop 배포를 지원합니다.
3.  **GitHub Pages**: 설정 > Pages에서 바로 배포 가능합니다.

### 데이터베이스(DB) 세팅에 대하여
요청하신 **"호스팅 서비스 DB 세팅"**과 관련하여 안내드립니다.

현재 이 애플리케이션은 **DB가 필요 없습니다.**
*   **이유**: 사용자가 파일을 올리면 즉시 분석해서 보여주고, 휘발되는 구조이기 때문입니다. 개인정보 보호를 위해 저장을 하지 않는 것이 더 유리합니다.

만약 **데이터 저장 기능(예: 회원가입, 내 성적 저장하기)**을 추가하고 싶다면, 다음 서비스를 추천합니다:

1.  **Firebase (Google)**: 가장 추천합니다. 서버 구축 없이 프론트엔드 코드만으로 DB(Firestore)를 사용할 수 있습니다.
2.  **Supabase**: Firebase의 오픈소스 대안으로, SQL(Postgres)을 쓰기 편합니다.

**만약 파일을 서버에 저장하고 싶다면?**
AWS S3 같은 스토리지 서비스가 필요하지만, 생기부 파일은 민감정보이므로 강력한 보안 조치(암호화 등)가 필수입니다. 따라서 현재의 **Client-Side** 방식을 유지하는 것을 강력히 권장합니다.

## 🛠 로컬 실행 방법

1. `site` 폴더로 이동합니다.
2. `index.html` 파일을 크롬 등 웹 브라우저로 엽니다.
3. 혹은 VS Code의 "Live Server" 확장을 사용하여 엽니다.
